# 向心力定量演示仪的传感器化改造

> 2026年广州市中小学实验教学能力展示活动参赛方案
> 把传统弹簧测力计定性演示升级为数字化定量探究，验证 F = mrω²

## What — 这是什么

高中物理实验教学的完整参赛方案，核心创新是用 **HX711 压力传感器 + 红外光电门 + ESP32 蓝牙** 替代传统弹簧测力计，实现向心力的数字化测量与定量验证。

包含：
- **实验运行模拟**（网页动画）：4画面同屏展示装置运转、ESP32数据流、手机APP、数据拟合的全链路
- **教学文稿**（Word）：七段式完整文稿，符合附件1-3排版规范
- **教学PPT**（两版）：设想图版 + 实拍留空版，15页16:9，含视频时间戳
- **ESP32固件**（Arduino）：HX711力传感器驱动 + 光电门中断测周期 + 蓝牙BLE传输
- **网页采集APP**：Web Serial API 连接ESP32，实时绘图 + 线性拟合 + 自动结论
- **3D打印模型**（STL + OpenSCAD源码）：旋转臂 + 滑块
- **装配工艺图**（7张）：从打印后处理到传感器标定的每一步
- **视频分镜脚本 + 文字讲稿 + 背景音乐清单**：15分钟教学视频的完整制作指南

## Required env

无环境变量要求。网页模拟和文档查看无需任何 API key。

ESP32固件烧录需要 Arduino IDE + ESP32 开发板支持包（见 docs/esp32_firmware/）。

## How to start

### 看实验运行模拟（最快）

直接打开 `src/index.html`，浏览器即可运行。4画面动画展示完整实验流程：
- 装置运转（3D转台旋转、光电门遮挡、力/ω/周期实时数值）
- ESP32数据流（HX711→ESP32→蓝牙波纹、JSON日志滚动）
- 手机APP（实时数值卡片 + F-t曲线动态绘制）
- 数据拟合（散点落点 + 最小二乘法拟合直线 + R²/误差实时计算）

播放控制：▶播放 / ⏸暂停 / ↺重置 / 1x·2x·4x速度 / 跳转各实验组

### 看完整文档

`docs/` 目录下所有 HTML 文件双击即可在浏览器打开：
- `材料尺寸清单.html` — 所有零件尺寸/材质/来源
- `零件制作工艺步骤.html` — 4阶段装配指南（配7张装配图）
- `调试与使用手册.html` — 标定流程 + 10条故障排查表
- `视频分镜脚本.html` — 13场景详细分镜
- `背景音乐清单.html` — 免版权配乐方案
- `采购清单.html` — 带淘宝搜索链接的采购表

Word/PPT 文件用 Office 打开。

### 自己动手做实验

1. 按 `docs/采购清单.html` 采购器材（约 ¥214）
2. 按 `docs/零件制作工艺步骤.html` 组装装置
3. 烧录 `docs/esp32_firmware/centrifugal_force.ino` 到 ESP32
4. 用 Chrome/Edge 打开 `docs/webapp/index.html` 连接 ESP32
5. 按 `docs/调试与使用手册.html` 标定传感器
6. 开始三组定量验证实验

## Troubleshooting

| 问题 | 解决 |
|---|---|
| 模拟动画打开空白 | 用 Chrome/Edge/Firefox，确保启用 JavaScript |
| ESP32连接失败 | 安装 CH340/CP2102 驱动；关闭其他串口软件；用 Chrome 87+ |
| 力读数为0或异常 | HX711 归零；检查 E+/E-/A+/A- 接线；重新标定 SCALE |
| ω始终为0 | 检查光电门 GPIO27 接线；手动拨旋转臂看是否有读数 |
| 拟合 R²<0.9 | 增加数据点至6个以上；删除明显偏离点；重新归零 |
| 3D打印件装配不上 | 中心孔用 Φ16 钻头扩孔（打印孔偏小0.1-0.3mm） |

详细故障排查见 `docs/调试与使用手册.html`。
