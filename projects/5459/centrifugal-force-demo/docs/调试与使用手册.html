方案A 向心力定量演示仪 - 调试与使用手册

一、首次使用流程

  1. 用Chrome或Edge浏览器打开 webapp/index.html（双击即可，无需联网）。
  2. ESP32用USB Type-C线连接电脑，电脑会识别为串口设备（Windows为COMx，Mac为/dev/cu.usbserial-xxx）。
  3. 点击网页「🔌连接ESP32」，浏览器弹出串口选择框，选择ESP32对应端口，波特率115200。
  4. 连接成功后状态显示「已连接」，力/ω/周期/转速四项实时刷新。
  5. 点「⚖️归零」，等待2秒（期间不要触碰传感器）。
  6. 挂100g砝码做力标定（见下方「传感器标定」章节）。
  7. 开始教学实验。

二、传感器标定（关键步骤）

2.1 力传感器标定

  1. 确保传感器无受力，点「归零」。
  2. 挂已知质量砝码（推荐100g，对应0.981N），等待读数稳定。
  3. 记录网页显示力值 F_read。若 F_read ≠ 0.981，需修正SCALE系数。
  4. 在网页调试栏（或串口监视器）发送：SCALE:新值，其中 新值 = 原值 × (0.981 / F_read)。
  5. 原值默认 -22000，可在串口发送 INFO 查询当前值。
  6. 重复2-3次，直到 F_read 误差 <2%。
标定只需做一次。SCALE值会保存在ESP32内存中，重启后恢复默认。若需永久保存，可修改固件中 SCALE_FACTOR 宏定义后重新烧录。

2.2 光电门标定

  1. 手动匀速转动旋转臂一圈，观察网页「转速」是否显示合理值（手动约30-60 r/min）。
  2. 若转速为0，检查光电门DOUT线是否接GPIO27、光电门缺口是否有遮挡物。
  3. 若转速数值异常大（如>1000），可能是滑块每圈遮挡多次，需调整光电门位置使其每圈只遮挡一次。

2.3 半径校准

  1. 用游标卡尺测量旋转中心（电机轴中心）到滑块中心的距离，即为半径r。
  2. 在网页「固定值」栏填入实际r值（单位cm，如 r=15）。
  3. 切换实验组时，固定值栏会自动解析，确保理论计算正确。

三、三组实验操作流程

3.1 F-ω² 组（固定 m、r，改变 ω）

  1. 固定值栏填 m=0.1,r=0.15（即100g、15cm），实验组选「F-ω²组」。
  2. 安装100g滑块，半径调至15cm。
  3. 电机调至最低速，等待转速稳定（网页ω值连续5秒波动<3%），点「记录数据点」。
  4. 电机调速提高一档，重复步骤3，共记录6组不同ω的数据。
  5. 点「拟合」，查看斜率是否接近 mr = 0.015，R²>0.95。

3.2 F-r 组（固定 m、ω，改变 r）

  1. 固定值栏填 m=0.1,w=5，实验组选「F-r组」。
  2. 保持100g滑块，电机调至ω≈5 rad/s 并保持调速器不动。
  3. 半径调至5cm，记录；再调至8、11、14、17、20cm各记录一次。
  4. 注意：改变半径时需停机，调整后再启动并等待转速恢复到5 rad/s。
  5. 拟合斜率应接近 mω² = 2.5。

3.3 F-m 组（固定 r、ω，改变 m）

  1. 固定值栏填 r=0.15,w=5，实验组选「F-m组」。
  2. 半径固定15cm，电机保持ω≈5 rad/s。
  3. 依次换装50g、75g、100g、125g、150g、175g滑块，每次记录。
  4. 拟合斜率应接近 rω² = 0.375。

四、数据导出与报告

  1. 实验完成后点「💾导出CSV」，文件名含时间戳。
  2. CSV包含序号、实验组、自变量x、力F四列，可用Excel打开。
  3. 在Excel中可自行作图，或直接用网页的拟合图截图插入实验报告。
  4. 网页「图象解释」栏的结论文字可直接复制到报告。

六、调试日志使用

  • 网页底部「🐞调试日志」显示所有操作记录与ESP32返回信息，带时间戳。
  • 发送 INFO 命令可查询ESP32当前零点、SCALE系数、中断计数。
  • 若数据异常，先看调试日志是否有 info 类消息提示。
  • 日志最多保留最近50条，超出自动滚动。

七、教学使用建议

  • 课前10分钟开机预热，完成力标定与归零。
  • 课堂演示流程：连接→归零→选实验组→采集→记录→拟合→解释，全程可投屏。
  • 学生分组实验时，每组用同一台ESP32+网页，轮流操作，数据各自导出。
  • 视频录制时，建议用OBS同时录制网页画面与实验装置近景，合成画中画。
技术支持：若按本手册无法解决问题，可在调试日志中截图，连同ESP32串口输出（串口监视器115200bps）一起反馈，便于定位。
