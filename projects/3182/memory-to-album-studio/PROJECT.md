# Memory to Album Studio

## What
Turn 5 life events into a bilingual 5-track concept album pack:
- Song titles
- Full CN+EN lyrics
- MV visual style notes
- Album concept + cover prompt

## Required env
None.

## How to start
Use Preview on this project directory as a static app.

## Outputs
- On-screen generated album package (copyable markdown)

## Troubleshooting
- If page is blank, refresh preview once.
- Keep exactly 5 events for best structure.
