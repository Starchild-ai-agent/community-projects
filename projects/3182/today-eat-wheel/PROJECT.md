## What

Today Eat Wheel is a lightweight web app that helps users decide what to eat by spinning a customizable wheel.

## Required env

None.

## How to start

This is a static web app.

- Serve `src/` as static files.
- Open `index.html` in a browser through your preview service.

## Outputs

- Interactive wheel with customizable options
- Randomized result selection with animated spin
- One-click copy of selected result

## Troubleshooting

- If the wheel appears blank, ensure `style.css` and `script.js` are in the same `src/` directory as `index.html`.
- If copy fails in browser, allow clipboard permission for the page.
