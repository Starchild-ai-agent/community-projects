Earnings Surprise Detector

v1.0.1 · MIT Python

What it does

Verifies actual quarterly earnings against analyst/research expectations BEFORE you act on a stock recommendation. Pulls real 扣非归母净利润同比 / 营收同比 / EPS via mx-data (东方财富权威数据), compares to the expected growth range, and gates the recommendation:

VerdictConditionAction
✅ PASSActual ≥ expected minimumProceed with recommendation
⚠️ WARNActual slightly below (within 20%)Size down 50%
❌ FAILActual significantly below or negativeExclude — do not recommend entry

Why it exists

Acting on unverified research expectations is expensive. Documented failures:

StockResearch ExpectedActualGap
太辰光 (300570)+80~120%-17%~100pp
英维克 (002837)+150%-82%~230pp
欧陆通 (300870)+180%Loss>180pp

In every case, the agent trusted research reports without pulling actuals. This skill makes verification mechanical: always pull, always compare, then act.

Quick start

pip install openpyxl
export MX_APIKEY=your_key_here

python3 verify_earnings.py \
  --stock "太辰光" \
  --code 300570 \
  --expected-min 80 \
  --expected-max 120 \
  --metric 扣非归母净利润同比

Exit codes: 0 = PASS, 1 = WARN, 2 = FAIL

Supported metrics

  • 扣非归母净利润同比 — default, most important for A-share verification
  • 营收同比 — revenue growth
  • 净利润同比 — headline profit (check vs 扣非 for earnings quality)
  • EPS — earnings per share

Required env

  • MX_APIKEY — 东方财富妙想 API key. Get it from dl.dfcfs.com
