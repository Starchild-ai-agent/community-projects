# 1382-alarm

## What
一个纯前端的闹钟网页应用，提供实时时钟显示、闹钟增删改开关、按周重复设置，以及触发时弹窗与蜂鸣提醒。

## Required env
无。

## How to start
这是静态服务项目，直接通过预览服务指向项目目录即可运行：

- 目录：`output/projects/1382-alarm/src`
- 端口：`9082`

## Outputs
- 网页界面（当前时间、日期）
- 闹钟列表（时间、标签、重复规则、开关状态）
- 到点提醒弹窗 + Web Audio 蜂鸣
- 本地持久化（`localStorage` 键：`alarms_v1`）

## Troubleshooting
- 如果没有声音：检查浏览器是否允许页面播放音频。
- 如果闹钟未触发：确认闹钟开关为开启，且系统时间与闹钟时间一致。
- 如果看不到历史闹钟：检查浏览器是否清除了本地存储数据。
