# Hello World Public Page

## What
一个最小可用的公开页面示例，用于演示如何把预览页面开源为 Public Project，并与已发布的公网 URL 关联。

## Required env
无必填环境变量。

## How to start
直接作为静态页面预览即可。

## Outputs
- `index.html`：页面入口文件

## Troubleshooting
- 如果页面无法公网访问，先确认预览已运行，再重新执行 publish_preview。
- 如果项目发布失败，先检查 `project.yaml` 字段是否完整且 `name` 与目录名一致。
