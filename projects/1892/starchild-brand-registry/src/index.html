The monolith

The monolith in the main logo represents the cognitive leaps of civilization: the moments when knowledge compounds, tools become infrastructure, and individual progress becomes shared progress.

Every user who creates an agent with Starchild improves their own knowledge system while also strengthening the knowledge of the network. Each agent becomes a new surface area for memory, workflows, taste, and capability to accumulate.

Typography

Power Grotesk

Use Power Grotesk for headlines, hero statements, section titles, and high-emphasis brand moments.

font-family: 'Power Grotesk', 'Google Sans', sans-serif;

Google Sans

Use Google Sans for body copy, UI text, labels, captions, tables, and product surfaces.

font-family: 'Google Sans', Inter, system-ui, sans-serif;

Colour system

Starchild’s primary colour represents energy, intuition, and momentum. Use it to signal key moments, actions, and emotional states.

The secondary orange palette adds depth and hierarchy. Use lighter values for backgrounds and gradients, darker values for emphasis and contrast.
