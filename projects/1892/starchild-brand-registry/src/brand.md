# Starchild Brand Guide

This is the shared brand context for Starchild agents.

Use this before creating images, slides, videos, landing pages, social posts, investor materials, dashboards, or any public-facing artifact.

## Brand assets

Primary asset folder: `brand/`

### Logos

Use files from:

- `brand/01_logos/primary/`
- `brand/01_logos/icon/`
- `brand/01_logos/wordmark/`
- `brand/01_logos/monochrome/`

Do not use files from `brand/01_logos/deprecated_do_not_use/` unless explicitly asked.

### Colors

Canonical color values live in:

- `brand/brand.json`
- `brand/02_colors/`

### Typography

Primary typeface for headings, body, and UI: Google Sans.

Power Grotesk is reserved for the Starchild logo and wordmark only.

Font files live in:

- `brand/03_typography/fonts/Google_Sans/GoogleSans-VariableFont_GRAD,opsz,wght.ttf`
- `brand/03_typography/fonts/Google_Sans/GoogleSans-Italic-VariableFont_GRAD,opsz,wght.ttf`

Canonical type rules live in:

- `brand/brand.json`
- `brand/03_typography/`

### Backgrounds

Use files from `brand/04_visual_system/backgrounds/`:

- `bg_black_grid.png`
- `BG_blackwithgreywords.jpg`
- `BG_monolith.png`
- `bg_orange_grid.png`
- `bg_orangewithblackwords.png`
- `bg_pixelburst.png`

### Templates

Use templates from:

- `brand/05_templates/slides/`
- `brand/05_templates/social/`
- `brand/05_templates/blog/`
- `brand/05_templates/video/`

## Brand voice

Clear, capable, direct, slightly opinionated. Avoid corporate filler, generic AI phrasing, overhype, and vague abstractions.

## Usage rule for agents

Before generating any brand-sensitive artifact, inspect `brand/brand.json` and the relevant subfolder. If required assets are missing, ask the user for the asset or create a clearly marked placeholder in the correct folder.
