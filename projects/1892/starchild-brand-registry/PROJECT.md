# Starchild Brand Registry

## What

A previewable brand guide and asset registry for Starchild. It gives humans and agents one place to reference logos, backgrounds, typography, colors, messaging, and implementation instructions.

## Required env

None.

## How to start

Serve `src/` as a static preview.

## Outputs

- `src/index.html`: visual brand guide
- `src/brand.json`: machine-readable agent instructions and asset paths
- `src/brand.md`: human-readable brand rules
- `src/tokens.css`: CSS token starter
- `src/assets/`: logos, web-optimized typography, and background assets

## Troubleshooting

If images do not load, check that asset paths are relative and the files exist under `src/assets/`.
