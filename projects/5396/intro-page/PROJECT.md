# intro-page

A clean personal intro page with light/dark theme toggle.

## What

Single-page HTML intro with name, role, bio, contact links, and a persistent theme toggle.

## Required env

None.

## How to start

Open `src/index.html` in a browser, or serve the `src/` directory with any static file server.

## Outputs

Static HTML page.

## Troubleshooting

Theme preference is stored in `localStorage`. Clear site data to reset.
