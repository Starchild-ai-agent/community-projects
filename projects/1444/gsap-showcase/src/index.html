skill · gsap-core

Tweens & Eases

GSAP 的最基本能力：把元素从 A 状态过渡到 B 状态。Easing 决定它怎么"感觉"运动。

gsap.to(".box", { x: 400, duration: 1.5, ease: "elastic.out(1, 0.3)" });
skill · gsap-core

Stagger · 错峰动画

一行代码让 100 个元素按顺序、按中心、按随机出场。

gsap.to(".cell", { scale: 0, stagger: { amount: 0.6, from: "center", grid: [10,10] } });

skill · gsap-performance

Performance · transform vs layout

左边动 left（触发 layout / paint），右边动 x（只 transform / GPU）。开 DevTools 看帧率差异。

❌ left / top（layout 重排）

left

✅ x / y（GPU transform）

x
// ❌ 慢：每帧重新计算 layout
gsap.to(".box", { left: 400 });

// ✅ 快：只走 compositor 线程
gsap.to(".box", { x: 400 });
skill · gsap-react / gsap-frameworks

框架集成

同样的 GSAP API，在 React / Vue / Svelte 里都能用。关键是清理。

⚛️ React (useGSAP)

import { useGSAP } from "@gsap/react";

useGSAP(() => {
  gsap.to(".box", { x: 100 });
}, { scope: containerRef });

🟢 Vue 3

onMounted(() => {
  ctx = gsap.context(() => {
    gsap.to(".box", { x: 100 });
  }, root.value);
});
onUnmounted(() => ctx.revert());

🟠 Svelte

onMount(() => {
  const ctx = gsap.context(() => {
    gsap.to(".box", { x: 100 });
  }, container);
  return () => ctx.revert();
});

怎么用上这些能力？

本页面所有动画背后，是 8 个独立的 GSAP skill —— 每一个都是一份让 AI Agent 写正确 GSAP 代码的"使用手册"。

📦 8 个 GSAP Skill 一览

  • gsap-core — Tweens, eases, stagger, matchMedia
  • gsap-timeline — 多步骤时间线编排
  • gsap-scrolltrigger — 滚动驱动 / pin / scrub
  • gsap-plugins — Flip / Draggable / MotionPath / SplitText 等
  • gsap-utils — clamp / mapRange / snap 数值帮手
  • gsap-react — useGSAP, React 集成
  • gsap-frameworks — Vue / Svelte 集成
  • gsap-performance — 性能最佳实践
