# 最小静态页面

A minimal static HTML page with modern dark theme styling.

## What

Zero-dependency single-page demo with gradient title, badge, and clean layout. Pure HTML + CSS, no JavaScript framework needed.

## Required env

None. This is a fully static site.

## How to start

Just open `src/index.html` in any browser, or serve via:

```
npx serve src/
```

## Outputs

A styled static page rendered at any static file server.

## Troubleshooting

- If styles don't render, ensure the browser supports CSS gradients (all modern browsers do)
- For preview in Starchild, use `preview(action="serve", dir="output/projects/minimal-static-page/src")`
