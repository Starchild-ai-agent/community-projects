# Run Lab

## What
Run Lab is a lightweight post-run analysis web app. It converts your run stats into actionable pace guidance, hydration targets, recovery timing, and the next session recommendation.

## Required env
None.

## How to start
Serve `src/` as a static site.

## Outputs
- Pace and speed summary
- Effort classification from heart rate/RPE
- Personalized hydration and carb/protein refill targets
- Suggested next workout and recovery timeline
- Copyable run debrief text

## Troubleshooting
- If numbers look off, ensure duration is total moving time in minutes.
- Enter distance in kilometers.
- Heart-rate based recommendations are best when average HR is provided.
