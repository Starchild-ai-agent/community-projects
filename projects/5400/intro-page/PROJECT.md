# Personal Intro Page

A minimal, responsive personal landing page.

## What

A single-page intro site with name, short bio, links, and contact. Built with vanilla HTML, CSS, and a small JavaScript interaction.

## Required env

None.

## How to start

Open `src/index.html` in any browser, or serve the `src` directory with a static file server on port 9080.

## Outputs

A public personal landing page served as a Starchild service.

## Troubleshooting

If styles don't load, ensure the `src` directory contains `index.html`, `styles.css`, and `script.js`.
