MONSKILLS EZ-Path

Agentic Monad Price Racing Orchestration of Diverse Aggregators

What this is

A TypeScript package that exposes EZ-Path DEX routing for agent workflows. It provides quote helpers and batch quote support across EVM routes.

Quick start

cd output/projects/monskills-ezpath/src/repo
npm install
npm run build
npm test
